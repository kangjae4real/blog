### 블로그에 신규기능을 추가했다.
지금 보고있는 블로그로 리뉴얼하기 전에는 게시글 댓글 기능이 존재하지 않았다. <br />
그래서일까. 글을 써도 반응이 없으니 블로깅을 하는 재미가 많이 줄어든 것 같았다.
<br />

### 그래서 추가했다.
[`Giscus`](https://github.com/giscus/giscus) 라는 Github Discussions을 기반으로 동작하는 댓글 서비스가 존재한다. <br />
서비스라기 보다는 간편하게 댓글 기능을 구현하게 도와주는 라이브러리이다. <br />
본인의 Github Repository를 Public으로 만들고 몇가지 Github Application을 <br />
활용하면 바로 댓글 기능을 개시할 수 있다.
<br />

### Giscus를 선택한 이유
간단히 댓글을 조회, 생성, 수정, 삭제를 하는 API를 개발하고 사용하는 방법도 정말 괜찮은 방법이겠지만 <br />
이미 너무 좋은 라이브러리들이 많으니 ㅎㅎ... <br />
과거에는 직접 만들어 쓰는 것이 최고라고 생각하던 때도 있었지만 회사생활을 근 4년을 하다보니 바뀐 나의 생각은 <br />
간편히 활용할 수 있는 서비스나 라이브러리들이 있다면 적극활용하는 것이 최고라고 생각하게 되었다. <br />
그 이유는 적은 개발 리소스를 사용해서 뚝딱 기능이 나와 시간을 아껴 동료 개발자를 도와주거나 <br />
다른 우선순위가 높은 업무를 보는것이 회사 입장에서나 개인 입장에서나 모두를 만족시킬 수 있다고 생각했기 때문이다.
<br />

### 결론
많이 댓글을 남겨주어 소통의 기회를 많이 얻고싶다. <br />
끝.
